# MyTracert

Аналог утилиты tracert (traceroute) для Windows, реализованный на C++ с использованием сырых сокетов и протокола ICMP.

## Возможности
- Определение маршрута до целевого узла (по IP-адресу или доменному имени)
- Отправка трёх ICMP-запросов на каждом шаге (TTL)
- Вывод времени отклика (RTT) для каждого пакета или `*` при потере
- Обратное DNS-разрешение (PTR) для промежуточных узлов
- Поддержка флагов командной строки:
  - `-d` — отключить обратные DNS-запросы
  - `-h <число>` — задать максимальное количество прыжков (по умолчанию 30)

## Установка (готовый установщик)
Все должно устанавливаться и открываться **от имени администратора**. Консоль cmd тоже **от имени администратора**.
Готовый установщик `MyTracertSetup.exe` доступен в корне репозитория. 
Запустите его от имени администратора — программа будет установлена в `C:\Program Files\MyTracert` и добавлена в системную переменную `PATH`. 
После этого команда `mytracert` доступна из любой командной строки. До ввода команды `mytracert` введите в консоль команду `chcp 65001` или `chcp 1251`.
